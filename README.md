# INFOGRAFIAS BLENDER

Landing page estatica para acceder a las infografias interactivas de Blender.

## Estructura

- `index.html`: pagina principal del menu.
- `assets/`: logotipos, botones, estilos y fondo topografico animado.
- `infografias/`: paginas interactivas enlazadas desde el menu.

Para publicarlo en GitHub Pages, sube esta carpeta completa al repositorio y usa `index.html` como pagina de inicio.
